# Portfolio Website

A minimal, clean personal portfolio built with React and Tailwind CSS.

## Features

- Minimal beige theme with dark mode support
- Fully responsive design
- Smooth animations with Framer Motion
- GitHub contribution graph integration
- Contact form
- Project showcase

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## Customization

1. Update your personal information in the components
2. Replace the placeholder profile image in `About.jsx`
3. Add your projects in `Projects.jsx`
4. Update your GitHub username in `GitHubGraph.jsx`
5. Update social links in `Contact.jsx`
6. Change "Your Name" throughout the components

## Technologies

- React 18
- Vite
- Tailwind CSS
- Framer Motion
