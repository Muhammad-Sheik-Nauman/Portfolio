Option 1: Screenshot This (Easiest)

Option 2: Use Canva (More Professional)

  1. Go to canva.com
  2. Create custom size: 1200 x 630 pixels
  3. Background color: #202023
  4. Add text: "Muhammad Sheik Nauman" (large, white)
  5. Add subtitle: "React & Frontend Developer" (smaller, #d9c8ab)
  6. Download as PNG
  7. Save to Portfolio/public/og-image.png

Option 3: Use This HTML File

  1. Open this file in Chrome/Edge
  2. Press F12 to open DevTools
  3. Click the device toolbar icon (or press Ctrl+Shift+M)
  4. Set dimensions to 1200 x 630
  5. Take a screenshot
  6. Save to Portfolio/public/og-image.png
